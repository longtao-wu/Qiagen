The Assignment¶

Take a ZIP file) of images and process them, using a library built into python that you need to learn how to use. A ZIP file takes several different files and compresses them, thus saving space, into one single file. The files in the ZIP file we provide are newspaper images (like you saw in week 3). Your task is to write python code which allows one to search through the images looking for the occurrences of keywords and faces. E.g. if you search for "pizza" it will return a contact sheet of all of the faces which were located on the newspaper page which mentions "pizza". This will test your ability to learn a new (library), your ability to use OpenCV to detect faces, your ability to use tesseract to do optical character recognition, and your ability to use PIL to composite images together into contact sheets.

Each page of the newspapers is saved as a single PNG image in a file called images.zip. These newspapers are in english, and contain a variety of stories, advertisements and images. Note: This file is fairly large (~200 MB) and may take some time to work with, I would encourage you to use small_img.zip for testing.

# Process the zipped source files, use dictionary here, and indexed by file name ('a-0.png' etc)
# Each image was saved in nested dictionary with three attributes

# {'file_name_1':{'raw_image': image_data, 'text': text_info, 'faces': list of faces}, 'file_name_2':{} ...}

src_images = {}

raw_files = zipfile.ZipFile('readonly/images.zip', 'r')

for item in raw_files.infolist():
    
    # print(item) # to show the processing details de-comment this line
    
    with raw_files.open(item) as file:
        
        # extract the image from zip file, convert to RGB mode
        image = Image.open(file).convert('RGB')
        
        # save the image data to dictionary 
        src_images[item.filename] = {'raw_image':image}
            
raw_files.close() # close the file before leaving

# Quick check by printout some information about these images

# load the face detection classifier provided by Coursera
face_cascade = cv.CascadeClassifier('readonly/haarcascade_frontalface_default.xml')

# Search for the faces in each page and define the boundary frame and extract information
for image_item in src_images.keys():
    
    # extract and convert the pixel information from image files
    image_opencv = np.array(src_images[image_item]['raw_image'])
    
    # Gandalf - the gray
    image_gray = cv.cvtColor(image_opencv, cv.COLOR_BGR2GRAY)
    
    # here is the major algorithm for face detection with common parameters 
    faces_collection = face_cascade.detectMultiScale(image_gray, 1.3, 5)
    
    src_images[image_item]['faces'] = [] # set up a empty 'faces' atrribute list
    
    for x, y, width, height in faces_collection:
        
        # Cut out each face detected by algorithm
        face = src_images[image_item]['raw_image'].crop((x, y, x + width, y + height))
        
        # Save them into our list
        src_images[image_item]['faces'].append(face)

#search the target, Christopher by default, in every page's text and return the faces on it

def search(target = 'Christopher'):
    
    # search each page/image
    for image_item in src_images.keys():

        # find our target
        if (target in src_images[image_item]['text']):

            # check whether faces are detected on the same image
            if(len(src_images[image_item]['faces']) != 0):
                
                print("Results found in file {}".format(image_item))
                
                # Maximal 5 faces in a row
                height = math.ceil(len(src_images[image_item]['faces'])/5)
                
                # The thumbnail maximized at 100x100 with .thumbnail function
                contact_sheet = Image.new('RGB',(500, 100 * height))
                
                x_axis = 0
                y_axis = 0
                
                for image_thumb in src_images[image_item]['faces']:
                    
                    contact_sheet.paste(image_thumb, (x_axis, y_axis))
                    
                    if x_axis + 100 == contact_sheet.width:
                        
                        x_axis = 0
                        y_axis += 100
                    
                    else:
                        x_axis += 100
                        
                display(contact_sheet)
                
            else:
                print("Results found in file {} \nBut there were no faces in that file!".format(image_item))
                
    return
